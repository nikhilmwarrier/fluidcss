Quick Start

Add this code to your <head> tag to get started right now using CDN. 

 <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"/>
  <link rel="stylesheet" href="https://nikhilmwarrier.github.io/fluidcss/fluid.css" />
  <script src="https://nikhilmwarrier.github.io/fluidcss/fluid.js" defer></script>

Starter template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this: 

<!DOCTYPE html>

  <html>

    <head>

      <!--Required meta tags-->  
      <meta charset="UTF-8">
      <meta name="viewport"
      content="width=device-width,
        initial-scale=1.0">

        <!--Google's Material Icons [Optional, but highly recommended]-->
        <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"
        />

        <!--FluidCSS Stylesheet-->
        <link rel="stylesheet" href="https://nikhilmwarrier.github.io/fluidcss/fluid.css" />

        <!--FluidCSS JavaScript-->
        <script src="https://nikhilmwarrier.github.io/fluidcss/fluid.js" defer></script>

      <title> FluidCSS Template </title>
    </head>


    <body>
      <div class="container">
        <h1>Hello World!</h1>
      </div>
    </body>

  </html>

A word on box-sizing

For more straightforward and digestible sizing, we use box-sizing: border-box on all elements. If you want to override this on some very specific cases, just add something like this to your css rules: 

some-element{
  box-sizing: content-box
}
