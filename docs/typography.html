Typography

Fonts

FluidCSS uses four default fonts:

Sans-serif:
Ubuntu and Roboto [Used in Buttons]

Serif:
Noto Serif

Monospace:
Fira Code

Note: These can be customized using the variables in FluidCSS :root section.

Inline Code

Feel free to use the <code> element.
Example usage:


<code>
    alert('this is how you use the code element') 
</code>

Output: alert('this is how you use the code element')

Note: Remember to replace < and > with &lt; &gt;, or it will be rendered as HTML and will not be displayed.

Code Blocks

Code blocks are created using the <pre class="code"><code> tags.
Example: 


<pre class="code" data-lang="HTML5"><code>
    alert('this is how you use code blocks') 
</code></pre>

Note: Setting the data-lang attribute on the >pre< element will show that language on the header of it's code block.

Also note: Be sure to keep code within <pre> tags as close to the left as possible; it will render all tabs.
