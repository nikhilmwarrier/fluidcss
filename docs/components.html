Modals

Pure CSS modals created using the :target pseudo element. Cool, right? 

<a href="#modal1" class="btn">Open Modal</a>   // Trigger to open modal

<div id="modal1" class="modal-overlay">    // The semitransparent overlay behind modal

<div class="modal">   
  <a href="#" title="Close" class="modal-close"> // Trigger to close modal
    &times;                                   <-- This creates the '×' symbol
  </a>
  <h1>Voilà!</h1>
  <p>
    Pure CSS Modal!
  </p>
</div>

</div>

Tooltips

Tooltips are used to provide more info about something

These can be made using: 

<span data-tooltip="Cool!">Hover over me!</span>

Example: Hover over me!
