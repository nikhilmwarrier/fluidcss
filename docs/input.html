User Inputs

Buttons

Usage: 

 <button class="btn"> //for primary button.
  <button class="btn btn-secondary"> //for secondary button. 
  <button class="btn" disabled>  //for disabled button 
  <button class="btn btn-secondary" disabled> //for disabled secondary button.

  //Pill Buttons
  <button class="btn btn-pill">Pill Button</button>
  <button class="btn btn-pill btn-secondary"> Secondary Pill Button</button>

  //Primary-colored Buttons
  <button class="btn btn-red">Red Button</button>       
  <button class="btn btn-green"> Green Button</button>
  <button class="btn btn-blue"> Blue Button</button>

  <button class="btn btn-block"> //for primary block-level (ie. 80% width) button. 
  <button class="btn btn-block btn-secondary"> //for secondary block-level button.





Checkboxes


 <label class="checkbox">                           <-- Create a label with a class of 'checkbox'
   <input type="checkbox" class="checkbox-input" /> <-- The actual checkbox, which is hidden
   <span class="checkbox-box"></span>               <-- The custom styled 'checkbox'
   Pizza                                            <-- Here, you can label your checkbox
 </label>
