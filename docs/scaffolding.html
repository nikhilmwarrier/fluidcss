Scaffolding

Layout System:

One of the key aspects of FluidCSS is that it has but a very small learning curve. It also removes the need of learning a gazillion class names just to position something.

FluidCSS has a very simple layout system. Just add three html tags to your body tag and, voila, the layout that took ages to make is ready!

The FluidCSS three-tag layout:

FluidCSS layout system is very simple. It comprises of a fixed sidebar, or <aside> element, a navbar at the top, created with the <nav> element, and the main <div class="container">, which is where your contemnt lies.

See the Pen FluidCSS in three tags by Nikhil M Warrier (@nikhilmwarrier) on CodePen.
